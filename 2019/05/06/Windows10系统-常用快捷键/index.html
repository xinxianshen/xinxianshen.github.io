Windows10系统-常用快捷键

发表于 2019-05-06 | | 阅读次数:

Win 组合键

Win + E：打开“我的电脑”

Win + R：打开“运行”对话框

Win + D：秒回桌面

Win + Home：最小化除当前窗口外的所有窗口

Win + Q：呼唤小娜

Win + L：瞬间锁屏

Win + A：打开通知栏

Win + Tab：虚拟桌面 （可用来快速切换工作窗口）

Win + X：打开所有系统设置”

Win + I：打开系统设置

Win + PauseBreak：打开系统属性

Win + -/+：屏幕放大或缩小 （让电脑屏幕拥有类似浏览器的放大缩小功能）

Win + ←/→：让对话窗口占左/右半屏 （在对比时非常有用）

Win + ↑/↓：让对话窗口放大/正常/最小化 （同上，先按左右再按上下可以让窗口只占角落1/4的区域）

PrintScreen：复制屏幕截图到粘贴板

Win + PrintScreen：复制屏幕截图到粘贴板，同时还会保存到 C:\Users\用户名\Pictures\Screenshots

Win + Shift + S：框选屏幕任意区域并复制到粘贴板

Ctrl 组合键

Ctrl + Z：撤回

Ctrl + Y：复原撤回

Ctrl + W：关闭

Ctrl + S：保存

Ctrl + N：新建窗口

Ctrl + Shift + N：新建一个文件夹

Ctrl + Shift + T：还原关闭的网页

Ctrl + Esc：显示开始菜单

Ctrl + Home：光标移至列首

Ctrl + Alt + Del：任务管理器

Ctrl + 选择文件：多选文件

Ctrl + 拖动文件： 复制文件

Ctrl + Shift + 任意箭头：突出显示一块文本

Ctrl + Shift + Esc：打开 Windows 任务管理器

Alt 组合键

Alt + F4：关闭当前窗口

Alt + Tab： 两个程序交换

Alt + F：打开程序菜单

Alt + 空格： 打开关闭对话窗口

Alt + 回车： 查看文件属性

Alt + 双击文件：查看文件属性

Alt + 拖动文件：创建快捷方式

Shift 组合键

Shift + 空格：半\全角切换

Shift + 回车：换行

Shift + Del：永久删除所选

Shift + F10：打开右键菜单

Shift + 选择文件：选择多个相邻的项目

Shift + 拖动文件： 移动文件
